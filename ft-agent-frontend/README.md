# FT-Agent 用户前端

财税智能 Agent 平台的用户界面，基于 Vue 3 + Element Plus。

## 功能

- 用户注册/登录
- 财税专家对话（基础版/专业版）
- 会话记忆管理
- 对话历史查看
- Token 余额和充值
- 订阅升级

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev
```

访问 http://localhost:3000

## 项目结构
